# Module 1: What Is Home Network Security?

Home network security is the process of protecting all devices in your home — phones, laptops, TVs, tablets, smart devices — from cyber threats.

## Why It Matters
- Prevent hacking
- Protect your identity
- Stop malware and scams
- Secure your kids’ devices

## Key Concepts
- Modem vs Router
- Network traffic
- LAN vs WAN
- IP addresses
